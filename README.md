#### Frontend:
```ReactJs and Redux```
```Styling TailwindCSS and DaisyUI```

#### Backend:
```Node.js```
```Database MongoDB```


#### to run locally:
```
$ git clone https://github.com/athi-v/chef-inventory-system.git
$ cd chef-inventory-system
$ npm install
$ cd frontend
$ npm install
$ cd ../
$ npm run dev
```
